# HTML 

## Introduction to HTML
### What is HTML? 

 HTML = Hyper Text Markup Language used to create the skeleton of the web page
 
 ### HTML Structure
 ```
 <!DOCTYPE html>
<html>
<head>
<title>Title of the Page</title>
</head>
<body>

<h1>H1 heading</h1>
<p>Paragraph content</p>

</body>
</html>
```
